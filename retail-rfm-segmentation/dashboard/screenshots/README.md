# Screenshots

Put your dashboard images here, for example:

- `dashboard_full.png` — the whole dashboard
- `segment_charts.png` — customers vs. revenue by segment
- `at_risk_table.png` — the retention call list
- `dashboard_demo.gif` — a short clip showing interactivity (click a segment,
  watch the visuals cross-filter)

Then reference them from the main `README.md`, e.g.:

```markdown
![Dashboard](dashboard/screenshots/dashboard_full.png)
```

(A GitHub README **cannot** embed a Power BI iframe, so a screenshot or GIF is
how the dashboard shows up in the repo. Keep the live embed on your portfolio.)
